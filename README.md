# 🎵 Offline Music Player

A modern, fast, and completely ad-free offline music player designed for smooth music playback without requiring an internet connection.

## Features

### 🎶 Music Playback
- Play local audio files
- Background playback support
- Shuffle and repeat modes
- Next and previous track controls
- Seek bar with smooth animations

### 📂 Library Management
- Automatic music scanning
- Browse songs, albums, artists, and folders
- Create and manage playlists
- Favorites collection

### 🎨 Modern UI
- Apple-inspired Liquid Glass design
- Dark Mode and Light Mode
- Smooth animations and transitions
- Responsive mobile interface
- Capsule-style controls

### 🔒 Privacy First
- No advertisements
- No tracking
- No data collection
- Works completely offline

### 🎧 Supported Formats
- MP3
- WAV
- AAC
- FLAC
- OGG
- M4A

## How It Works

1. Install the application.
2. Grant storage/media access permission.
3. The app automatically scans your device for music files.
4. Browse and play your favorite songs.
5. Create playlists and enjoy music without interruptions.
